# devbox

### setup
```bash
# installs home_brew and sets some lovely env vars
./setup.sh init_mac
# installs some dev, build tools, along with SpaceVim
./setup.sh init_dev_tools
# installs some sdks
./setup.sh init_sdks
```

### config
```bash
./setup.sh config_box
./setup.sh config_prompt
```

### binaries

```bash
npm install .
npm link
```

dev -h          # mainframe
